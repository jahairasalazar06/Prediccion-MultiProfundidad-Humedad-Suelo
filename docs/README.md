# Predicción Multi-Profundidad de la Humedad del Suelo

Dashboard interactivo desarrollado en Python para la visualización y análisis de la humedad del suelo mediante modelos de aprendizaje automático supervisado.

## Objetivo
Desarrollar un modelo predictivo que permita estimar la humedad del suelo a 60 cm y 100 cm utilizando mediciones superficiales (20 cm) y variables hidrometeorológicas, reduciendo la necesidad de instalar sensores en múltiples profundidades.

## Tecnologías utilizadas
- Python
- Google Colab
- Pandas
- Plotly
- Folium
- Scikit-learn
- XGBoost
- GitHub Pages
